使用指南 - Minecraft Command Editor Wiki

一、系统要求

请尽可能的在 Windows 10 上使用 Minecraft Command Editor ，以确保 Minecraft Command Editor 能发挥出最大的性能。

1.1 支持库

  • 当你使用 Minecraft Command Editor 时，你可能需要以下列出的支持库：
    • Microsoft .NET Framework 4.6.1 或更高版本
    • Microsoft.ACE.OLEDB.12.0
  • 请务必检查你的计算机是否安装了以上支持库，若没有安装，请通过 1.2 安装帮助 的引导完成安装。

1.2 安装帮助

  • 1.2.1 安装 Microsoft .NET Framework（离线安装）
      1. 打开 MCE ；
      2. 进入 设置 - 支持库 ；
      3. 点击 安装 .NET Framework 4.7.2 按钮即可进行安装。
  • 1.2.2 安装 Microsoft.ACE.OLEDB.12.0（离线安装）
      1. 如果是第一次运行 MCE ，将会弹出安装 Microsoft.ACE.OLEDB.12.0 支持库的窗口，直接安装即可；
      2. 如果不是第一次运行，则打开 MCE ；
      3. 进入 设置 - 支持库 ；
      4. 根据你的电脑的系统类型，点击 安装 AccessDatabaseEngine 或 安装 AccessDatabaseEngine_X64 按钮，选择 32 位或 64 位的支持库进行安装。
  • 1.2.3 安装 Microsoft .NET Framework（在线安装）
      1. 登录 Microsoft .NET 服务官网（dotnet.microsoft.com）；
      2. 点击 Download ；
      3. 选择合适的版本和语言进行下载。

二、错误解决方案

2.1 支持库类错误

  • 2.1.1 由于方法 'System.Reflection.MethodInfo.CreateDelegate' 的缺失而导致程序组件发生异常
    • 错误代码：
    • Unhandled exception has occurred in a component in your application. If you click Continue, the application will ignore this error and attempt to continue.

      Method not found: 'System Delegate System.Reflection.MethodInfo.CreateDelegate(System.Type)'
    • 解决方案：
    • 尝试使用 .NET Framework 4.5 或更高版本，推荐使用 .NET Framework 4.6.1 或更高版本。
  • 2.1.2 由于支持库 'Microsoft.ACE.OLEDB.12.0' 未在计算机上注册而导致程序发生异常
    • 错误代码：
    • Unhandled exception has occurred in your application. If you click Continue, the application will ignore this error and attempt to continue. If you click Quit, the application will close immediately.

      The 'Microsoft.ACE.OLEDB.12.0' provider is not registered on the local machine.
    • 解决方案：
    • 重新安装 AccessDatabaseEngine 或 AccessDatabaseEngine_64X ，可以在 '设置 - 支持库' 中进行安装。

1.2 操作类错误

  • 2.2.1 错误代码：GiveItems0001
    • 检查是否已选择了要获取物品的玩家。
  • 2.2.2 错误代码：GiveItems0002
    • 检查是否已正确填写了要获取物品的玩家 ID。
  • 2.2.3 错误代码：GiveItems0003
    • 检查是否已选择了要获取的物品类别。
  • 2.2.4 错误代码：GiveItems0004
    • 检查是否已选择了要获取的物品。
  • 2.2.5 错误代码：GiveItems0005
    • 检查是否已选择了要获取的物品。
  • 2.2.6 错误代码：StatusEffect0001
    • 检查是否已选择了要获取状态效果的玩家。
  • 2.2.7 错误代码：StatusEffect0002
    • 检查是否已正确填写了要获取状态效果的玩家 ID。
  • 2.2.8 错误代码：StatusEffect0003
    • 检查是否已选择了要获取的状态效果。
