使用指南 - Minecraft Command Editor Wiki

一、系統要求

請盡可能的在 Windows 10 上使用 Minecraft Command Editor ，以確保 Minecraft Command Editor 能發揮出最大的性能。

1.1 支持庫

  • 當你使用 Minecraft Command Editor 時，你可能需要以下列出的支持庫：
    • Microsoft .NET Framework 4.6.1 或更高版本
    • Microsoft.ACE.OLEDB.12.0
  • 請務必檢查你的計算機是否安裝了以上支持庫，若沒有安裝，請通過 1.2 安裝幫助 的引導完成安裝。

1.2 安裝幫助

  • 1.2.1 安裝 Microsoft .NET Framework（離線安裝）
      1. 打開 MCE ；
      2. 進入 設置 - 支持庫 ；
      3. 點擊 安裝 .NET Framework 4.7.2 按鈕即可進行安裝。
  • 1.2.2 安裝 Microsoft.ACE.OLEDB.12.0（離線安裝）
      1. 如果是第一次啟動 MCE ，將會彈出安裝 Microsoft.ACE.OLEDB.12.0 支持庫的窗口，直接安裝即可；
      2. 如果不是第一次啟動，則打開 MCE ；
      3. 進入 設置 - 支持庫 ；
      4. 根據你的計算機的系統類型，點擊 安裝 AccessDatabaseEngine 或 安裝 AccessDatabaseEngine_X64 按鈕，選擇 32 位或 64 位的支持庫進行安裝。
  • 1.2.3 安裝 Microsoft .NET Framework（在线安装）
      1. 登錄 Microsoft .NET 服務官網（dotnet.microsoft.com）；
      2. 點擊 Download ；
      3. 選擇合適的版本和語言進行下載。

二、錯誤解決方案

2.1 支持庫類錯誤

  • 2.1.1 由於方法 'System.Reflection.MethodInfo.CreateDelegate' 的缺失而導致程序組件發生異常
    • 錯誤代碼：
    • Unhandled exception has occurred in a component in your application. If you click Continue, the application will ignore this error and attempt to continue.

      Method not found: 'System Delegate System.Reflection.MethodInfo.CreateDelegate(System.Type)'
    • 解決方案：
    • 嘗試使用 .NET Framework 4.5 或更高版本，推薦使用 .NET Framework 4.6.1 或更高版本。
  • 2.1.2 由於支持庫 'Microsoft.ACE.OLEDB.12.0' 未在計算機上註冊而導致程序發生異常
    • 錯誤代碼：
    • Unhandled exception has occurred in your application. If you click Continue, the application will ignore this error and attempt to continue. If you click Quit, the application will close immediately.

      The 'Microsoft.ACE.OLEDB.12.0' provider is not registered on the local machine.
    • 解決方案：
    • 重新安裝 AccessDatabaseEngine 或 AccessDatabaseEngine_64X ，可以在 '設置 - 支持庫' 中進行安裝。

1.2 操作類錯誤

  • 2.2.1 錯誤代碼：GiveItems0001
    • 檢查是否已選擇了要獲取物品的玩家。
  • 2.2.2 錯誤代碼：GiveItems0002
    • 檢查是否已正確填寫了要獲取物品的玩家 ID。
  • 2.2.3 錯誤代碼：GiveItems0003
    • 檢查是否已選擇了要獲取的物品類別。
  • 2.2.4 錯誤代碼：GiveItems0004
    • 檢查是否已選擇了要獲取的物品。
  • 2.2.5 錯誤代碼：GiveItems0005
    • 檢查是否已選擇了要獲取的物品。
  • 2.2.6 錯誤代碼：StatusEffect0001
    • 檢查是否已選擇了要獲取狀態效果的玩家。
  • 2.2.7 錯誤代碼：StatusEffect0002
    • 檢查是否已正確填寫了要獲取狀態效果的玩家 ID。
  • 2.2.8 錯誤代碼：StatusEffect0003
    • 檢查是否已選擇了要獲取的狀態效果。
